# 🌸 Support Vector Machine (SVM) on Iris Dataset

This project demonstrates how to use **Support Vector Machines (SVM)** for classification using the famous **Iris dataset**. It compares different kernel functions to evaluate their performance.

---

## 📌 Project Overview

The goal of this project is to:

- Understand how SVM works for classification
- Compare different SVM kernels:
  - Linear
  - Polynomial
  - Radial Basis Function (RBF)
- Visualize decision boundaries
- Evaluate model performance

---

## 📊 Dataset

The project uses the built-in **Iris dataset** from `sklearn.datasets`.

### Features:
- Sepal length
- Sepal width
- Petal length
- Petal width

### Classes:
- Setosa
- Versicolor
- Virginica
